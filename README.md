# Expense-Tracker-Web-App-JavaScript-
Built an Expense Tracker Web App using HTML, CSS &amp; JavaScript with LocalStorage support to manage daily expenses and improve my frontend development skills.

A simple and responsive Expense Tracker web application built using HTML, CSS and JavaScript.
This app allows users to add, view and manage daily expenses and stores data directly in the browser using LocalStorage.

It helps users track spending in an easy and organized way while showcasing my frontend and JavaScript skills.

Key Features :

Add expense with amount, category & date

Data saved in browser (LocalStorage)

Clean and user-friendly UI

Fully client-side project

Beginner-friendly and lightweight
